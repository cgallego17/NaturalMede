# Logo de NaturalMede

Para usar el logo real de NaturalMede, sigue estos pasos:

## 1. Preparar la imagen
- Renombra tu archivo de logo como `logo.png`
- Asegúrate de que la imagen sea cuadrada (recomendado: 200x200 píxeles o más)
- El formato debe ser PNG para mantener la transparencia

## 2. Subir el archivo
- Coloca el archivo `logo.png` en esta carpeta: `static/images/`
- El archivo debe llamarse exactamente `logo.png`

## 3. Verificar
- El logo aparecerá automáticamente en:
  - Sidebar del admin personalizado (120x120px)
  - Navbar del sitio principal (60x60px)
- Ambos logos tendrán forma circular con borde crema

## Estructura de archivos esperada:
```
static/
└── images/
    └── logo.png  ← Tu logo aquí
```

## Notas técnicas:
- Los logos se redimensionan automáticamente
- Se aplica un efecto hover (escala 1.05)
- Bordes cremas para mantener la identidad visual
- Sombras para profundidad visual
